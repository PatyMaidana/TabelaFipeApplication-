# CONSULTA TABELA FIPE
Essa aplicação permite a consulta de veículos, por tipo, marca e modelo. 
Através dela é possível comparar os valores de um mesmo modelo, com anos de fabricação diferentes. 
Objetivo dessa aplicação - projeto acadêmico - desafio Alura. 

## Linguagem 
Java - Spring Boot
